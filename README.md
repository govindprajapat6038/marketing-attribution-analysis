# Marketing Attribution Analysis

## Project Overview
This project analyzes marketing campaign performance using SQL.

## Dataset
- Marketing_Dataset_150_Rows.xlsx

## SQL Analysis Performed
1. Total Revenue by Channel
2. Total Cost by Channel
3. Total Conversions by Channel
4. ROAS (Return on Ad Spend) by Channel

## Key Findings
- Instagram generated the highest revenue.
- Facebook achieved the highest ROAS.
- Email had the lowest ROAS.
