# Marketing Attribution Analysis

## Project Overview
This project analyzes marketing campaign performance using SQL. The goal is to identify the most effective marketing channels based on revenue, conversions, clicks, profit, and return on ad spend (ROAS).

## Dataset
- Marketing_Dataset_150_Rows.xlsx
- Imported into MySQL as `marketing_data`

## Tools Used
- MySQL Workbench
- SQL
- GitHub

## SQL Analysis Performed

### 1. Total Revenue by Channel
Identified which marketing channel generated the highest revenue.

### 2. Total Conversions by Channel
Compared conversions across all channels.

### 3. Total Cost by Channel
Analyzed advertising spend by channel.

### 4. ROAS (Return on Ad Spend)
Calculated revenue generated for every unit of advertising cost.

### 5. Top 5 Campaigns by Revenue
Identified the highest-performing campaigns.

### 6. Conversion Rate by Channel
Measured the percentage of clicks converted into customers.

### 7. Cost Per Conversion
Calculated the average cost required to generate one conversion.

### 8. Profit by Channel
Profit = Revenue - Cost

### 9. Average Revenue by Channel
Compared average campaign revenue across channels.

### 10. Total Clicks by Channel
Measured user engagement for each channel.

## Key Findings

- Instagram generated the highest revenue (758,353).
- Facebook achieved the highest ROAS (3.11).
- Google Ads generated the highest clicks (38,028).
- Facebook achieved the highest conversion rate (11.23%).
- Email had the lowest ROAS among all channels.
- Instagram generated the highest profit (507,777).
